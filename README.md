# Agenda Vital & Didáctica

Una aplicación web diseñada para ser una agenda personal amigable, con herramientas de bienestar emocional como:

- **Dashboard**: Vista general con saludo y accesos rápidos.
- **Agenda**: Gestión de tareas diarias.
- **Jardín Reflexivo**: Espacio para registrar pensamientos y emociones.
- **Arte Terapia**: Lienzo digital para dibujar y relajarse.
- **Centro de Calma**: Ejercicios de respiración y relajación.

## Despliegue en Vercel

Esta aplicación está lista para ser desplegada como un sitio estático.

1.  Ve a [Vercel](https://vercel.com) e inicia sesión.
2.  Haz clic en **"Add New..."** -> **"Project"**.
3.  Selecciona el repositorio `agenda-app` de tu lista.
4.  En la configuración del proyecto, deja todo por defecto (Framework Preset: **Other**).
5.  Haz clic en **Deploy**.

¡Listo! Vercel te dará una URL pública para acceder a tu agenda.

## Ejecución Local

Para probar la aplicación en tu computadora:

1.  Asegúrate de tener Python instalado.
2.  Ejecuta el script `start_app.bat` (en Windows).
3.  O corre manualmente: `python server.py`
4.  Abre `http://localhost:8081` en tu navegador.
